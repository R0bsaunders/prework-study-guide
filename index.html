GitHub

  • Cheat Sheet
  • 'git status': checks what branch we are currently on
  • 'git checkout -b branch-name': creates a new branch and switches to it
  • 'git add -A' to commit all files in the working branch.
  • 'git commit -m "message here"' associate a message with our commit. Commit messages should be short descriptive messages of the changes implemented
  • git pull origin main to check all is up to date
  • Pwd – displays current directory
  • Ls – displays list of files and folders in current directory
  • Cd – change directory
  • Cd .. – go back one folder
  • Mkdir – creates a directory (folder)
  • Touch – creates a file
  • Start – opens the file
  • An SSH key is an access credential for the secure shell (SSH) network protocol. Generating SSH keys allows developers to interface with certain remote services without having to constantly retype login information.
